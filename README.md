## 🤡 ¿Qué hace esta abominación?
OQE es una aplicación Spring Boot MVC falsa que parodia sin piedad:
- El diseño web moderno "Vende Humo" (Dark mode exagerado, gradientes innecesarios, Glassmorphism donde no se necesita).
- Las expectativas irreales y rídiculas del reclutamiento IT en 2026.
- El uso excesivo de buzzwords por parte de "Agencias Tech" y Startups que realmente solo son un wrapper a la API de OpenAI.
- Los infames "Contratos de Renuncia Voluntaria al Bienestar Físico y Emocional" disfrazados de "perks" como *Pizza Fridays* y pago en *Equity*.

## 🛠️ Stack Tecnológico (Destructivo)
- **Backend:** Java 21 + Spring Boot (Usado principalmente para renderizar HTML porque amamos el sobre-ingenio).
- **Frontend:** Thymeleaf + Bootstrap 5. "Optimizamos" Bootstrap sobrescribiéndolo con CSS puro que simula vórtices cósmicos de inversores.
- **Micro-Animaciones:** CSS puro. Tan eficientes que aceleran tu GPU para simular que estás minando crypto.
- **Lógica de FOMO:** Vanilla JS. Implementamos notificaciones falsas ("Tú ex-compañero acaba de ser promovido por robarse tu idea") para inyectarte ansiedad en tiempo real.

## 🚀 Cómo correr esto (Bajo tu propio riesgo psicológico)
1. Asegúrate de tener al menos 32GB de RAM (es broma, con 2GB basta, pero así lo diría un anuncio real).
2. Clona el repositorio.
3. Ejecuta el monolito descentralizado:
   ```bash
   ./mvnw spring-boot:run
   ```

4. Abre `http://localhost:8080/` en tu navegador.
